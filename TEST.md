## 🚀 快速开始

### 启动服务器
```bash
# 安装依赖
npm install

# 启动服务器
npm start

# 访问应用
open http://localhost:3000
```

### 使用流程

1. **创建房间**
   - 访问主页 http://localhost:3000
   - 点击"🔗 Create New Room"
   - 系统生成唯一房间链接

2. **分享链接**
   - 复制房间链接或房间码
   - 通过邮件、社交媒体分享
   - 其他用户点击链接加入房间

3. **传输文件**
   - 所有用户访问同一房间链接
   - 拖拽文件到上传区域或点击选择
   - 实时查看上传进度
   - 上传完成后可下载文件

## 🔧 技术架构

### 前端技术栈
- **HTML5**：语义化标记
- **CSS3**：响应式设计、动画效果
- **JavaScript (ES6+)**：现代化JavaScript
- **Socket.IO Client**：实时通信

### 后端技术栈
- **Node.js**：运行时环境
- **Express**：Web框架
- **Socket.IO**：实时通信库
- **Multer**：文件上传中间件

### 数据流
```
用户上传 → Multer处理 → Socket.IO广播 → 进度更新 → 完成通知
```

## 📝 常见问题

### Q: 文件上传失败怎么办？
A: 检查文件大小是否超过50MB，网络连接是否正常。

### Q: 进度条不更新？
A: 可能是网络延迟，稍等片刻或刷新页面。

### Q: 移动端无法上传？
A: 确保使用现代浏览器，部分老旧浏览器可能不支持。

### Q: 如何提高传输速度？
A: 使用有线网络，关闭其他占用带宽的应用。
