## 🚀 如何使用

### 启动服务器
```bash
# 1. 安装依赖
npm install

# 2. 启动服务器
npm start

# 3. 访问应用
open http://localhost:3000
```

### 使用流程
1. **创建房间**：点击"Create New Room"
2. **分享链接**：复制房间链接给其他人
3. **传输文件**：
   - 拖拽文件上传
   - 复制文件后 Ctrl+V 粘贴
   - 截图后 Ctrl+V 粘贴
4. **预览文件**：
   - 图片：查看缩略图，点击预览
   - 视频：内嵌播放器，支持控制
   - 文本：显示前10行，支持复制和展开
5. **享受Apple风格UI**：
   - 浅灰色背景
   - 白色卡片设计
   - 胶囊按钮
   - 毛玻璃效果
   - 平滑动画
