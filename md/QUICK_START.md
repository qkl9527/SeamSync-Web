## 🚀 立即体验

### 1. 启动服务器（如果还没启动）
```bash
cd /Users/qikl/qikl/wpt/dev/front/claude-code/claude-demo5-websync
npm install
npm start
```

### 2. 打开浏览器
访问：**http://localhost:3000**

### 3. 创建房间
- 点击"🔗 Create New Room"
- 复制生成的链接

### 4. 分享给朋友
- 将链接发送给其他人
- 大家同时访问同一链接

### 5. 开始传输文件
**方式一：拖拽上传**
- 直接拖拽文件到上传区域

**方式二：选择文件**
- 点击"Browse Files"选择文件

**方式三：粘贴文件（推荐！）**
- 复制文件后按 `Ctrl+V` (Windows/Linux) 或 `Cmd+V` (Mac)
- 截图后直接 `Ctrl+V` 粘贴
- 复制文本后 `Ctrl+V` 粘贴

### 6. 预览文件
- **图片**：查看缩略图，点击"👁️ Preview"看大图
- **视频**：内嵌播放器，点击"👁️ Preview"全屏播放
- **文本**：显示前10行，点击"📋 Copy"复制，"📖 View All"查看全部

## 🆘 常见问题

### Q: 服务器启动失败？
A: 确保端口3000未被占用，或修改 `.env` 文件更改端口

### Q: 粘贴功能不工作？
A: 确保使用现代浏览器（Chrome 120+, Firefox 110+, Safari 15+）

### Q: 预览不显示？
A: 检查文件类型是否支持，图片/视频需要浏览器支持

### Q: 深色模式不切换？
A: 确保系统开启了深色模式，浏览器会自动适配
